1. The Core Model (XGBoost)

Driven by an XGBoost Regressor on AWS EC2. It maps weather vectors (Humidity, Pressure, Wind) to Irradiance flux with sub-50ms latency.

2. Solar Physics (PVLib)

The Ineichen Clearsky Model calculates the precise Solar Zenith Angle for the Pincode's coordinates to establish a physics-based baseline.

3. Deployment Architecture

This project is live via a Decoupled Cloud Architecture: The UI is hosted on AWS S3, while the AI logic runs on an Ubuntu 24.04 EC2 instance managed by PM2.
